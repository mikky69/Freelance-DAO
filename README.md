# Freelance DAO  

**Freelance DAO** is a decentralized freelance marketplace built on the **Internet Computer blockchain**, connecting clients and freelancers in a trustless, transparent, and efficient ecosystem.  

## 🌟 Features  
- 🔗 **Blockchain-Powered** – Secure, trustless transactions via smart contracts.  
- 💰 **Low Fees** – Reduced platform fees compared to traditional freelance marketplaces.  
- 🤝 **Fair Dispute Resolution** – Decentralized governance for handling disputes.  
- 🌍 **Global Access** – Open to freelancers and clients worldwide.  

## 🛠️ Tech Stack  
- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** Internet Computer (Motoko, Rust)  
- **DevOps:** GitHub Actions, CI/CD, Docker  

## 🚀 Getting Started  

### **1️⃣ Clone the Repository**  
```bash
git clone https://github.com/Freelance-DAO/freelance-dao.git
cd freelance-dao
``` 

### **2️⃣ Install Dependencies**
npm run dev  # Start frontend
dfx deploy   # Deploy smart contracts
